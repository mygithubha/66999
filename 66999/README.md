# 图片领取系统

这是一个简单的图片领取管理系统，采用PHP + MySQL开发。

## 功能特点

- 前台：
  - 组别选择
  - 图片领取
  - 领取记录显示
  - 防止重复领取（基于IP和UA）
  - 图片预览和保存

- 后台：
  - 管理员登录
  - 组别管理（增删改查）
  - 图片上传
  - 领取状态查看

## 安装说明

1. 确保您的服务器已安装PHP 7.0+和MySQL 5.7+
2. 将所有文件上传到网站根目录
3. 创建数据库并导入`database.sql`文件
4. 修改`config.php`中的数据库连接信息
5. 确保`uploads`目录具有写入权限

## 默认登录信息

- 后台地址：`/admin.php`
- 用户名：`admin`
- 密码：`admin123`

## 注意事项

- 请及时修改默认管理员密码
- 定期清理uploads目录中的无用图片
- 建议设置上传图片的大小限制
